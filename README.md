A `README.md` a projekt belépési pontja: megmagyarázza, hogy mire való az adott repo, hogyan kell telepíteni, konfigurálni és futtatni. GitHub és GitLab automatikusan rendereli a repo főoldalán, így ez az első dolog, amit egy új fejlesztő vagy látogató lát.

A **Markdown** (`.md`) egy egyszerű, emberi szemmel is olvasható szövegformázási nyelv, amelyet John Gruber alkotott 2004-ben. Célja, hogy minimális szintaxissal lehessen strukturált, HTML-be konvertálható dokumentumokat írni — szövegszerkesztő sem kell hozzá, elég egy egyszerű editor.

**Legfontosabb markerek:**

- `# H1`, `## H2`, `### H3` — fejlécszintek
- `**félkövér**`, `*dőlt*` — kiemelések
- `-` vagy `1.` — felsorolás / számozott lista
- `` `kód` `` — inline kód, ` ```blokk``` ` — kódblokk (nyelvvel: ` ```python `)
- `[szöveg](url)` — hivatkozás
- `![alt](url)` — kép
- `---` — vízszintes elválasztó
- `> szöveg` — idézet (blockquote)
- `| col1 | col2 |` — táblázat
